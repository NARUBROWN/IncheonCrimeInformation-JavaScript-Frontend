# IncheonCrimeInformation-JavaScript-Frontend
인천재능대학교 인공지능컴퓨터정보과 파이썬빅데이터수집 전공과목 시간에 진행했던 <br>'인천광역시 지자체 통계를 통하여 바라본 방범용 CCTV 시설과 범죄발생률의 상관관계' 프로젝트 입니다.
